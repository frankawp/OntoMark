# 知识库本体定义

> 新闻类多领域知识库，涵盖科技、体育、娱乐、商业等领域的实体和事件追踪。

## 知识维度

### Actor / 行动者

> 新闻中涉及的个人和组织行动者。记者、运动员、CEO、艺人等。

- **粒度**：标准
- **属性**：
  - 角色/职位
  - 所属组织
  - 国籍

### Event / 事件

> 新闻报道的核心事件。比赛、产品发布、争议、交易、会议等。

- **粒度**：详细
- **属性**：
  - 时间
  - 地点
  - 参与方
  - 结果

### Thing / 事物

> 被讨论的产品和技术。游戏、设备、电影、AI 模型等。

- **粒度**：标准
- **属性**：
  - 类型
  - 开发者/创造者
  - 用途

### Organization / 组织

> 公司和机构。OpenAI、Apple、Google、Epic Games、政府机构等。

- **粒度**：标准
- **属性**：
  - 类型
  - 行业
  - 总部

### Topic / 主题

> 知识分类和聚合标签。Sports、Technology、Health 等。

- **粒度**：粗略
- **属性**：
  - 关联维度

---

## 维度关联

- Actor **参与** Event
- Actor **属于** Organization
- Organization **创造** Thing
- Event **涉及** Thing
- Event **分类为** Topic
- Topic **包含** 多个实体
